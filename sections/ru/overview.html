1 Обзор

Модуль регистрации пациентов является центральным компонентом системы MediMind EMR, который позволяет:

  • Регистрировать новых пациентов в клинике
  • Искать существующих пациентов по множеству критериев
  • Изменять данные пациента по мере необходимости
  • Открывать новые визиты для зарегистрированных пациентов

Модуль предназначен для сотрудников ресепшн и медицинских регистраторов.
