2 Функции

Система поиска

Система предлагает гибкий поиск по множеству параметров:

Тип поиска Описание FHIR параметр
Быстрый поиск По личному номеру или имени — автозаполнение identifier | name
Детальный поиск Множество фильтров: дата, отделение, статус _lastUpdated | organization

4 секции формы

Форма регистрации разделена на 4 логические секции:

Варианты сохранения

Сохранить
Только сохранение данных пациента
Сохранить и продолжить
Регистрируется, открываются шаги регистрации визита
Сохранить и новый
Регистрируется, форма очищается для нового пациента
Сохранить и просмотреть
Регистрируется, открывается карточка пациента

Шаги регистрации визита

1
Детали регистрации
Тип визита, дата, отделение
2
Данные страхования
Страховая компания, полис
3
Гарантия
Гарантийное письмо, сумма
4
Демография
Регион, диагноз, жалоба
5
Подтверждение
Сводка и регистрация

Управление страхованием

Система позволяет добавить до 3 страховых компаний на один визит.
