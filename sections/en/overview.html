2 Overview

The Patient Registration module is a central component of the MediMind EMR system that allows you to:

  • Search patients using unified hero search with keyboard shortcuts (Cmd/Ctrl+K)
  • Register new patients with comprehensive data collection across 8 form sections
  • Create visits optionally as part of patient registration
  • View today's appointments and recent patients in the desktop sidebar
  • Upload documents including guarantee letters with attachments
  • Auto-save drafts to prevent data loss during registration

The module is designed for reception staff and medical registrars, with a responsive interface that adapts to desktop and mobile devices.

Unified Patient Search
Hero search with Cmd/Ctrl+K shortcut, auto-detection of ID vs name, and advanced filters
Patient Lookup
11-digit personal ID lookup with real-time validation and patient card display
Draft Auto-Save
localStorage persistence with 1-second debounce prevents data loss
Mobile Wizard
5-step responsive wizard for screens under 768px width
Document Upload
Drag & drop file upload for guarantee letters, up to 5 files per guarantee
Today's Visits
Desktop sidebar showing current day's encounters with status badges

Component Architecture

The registration module uses a modern React component architecture:
