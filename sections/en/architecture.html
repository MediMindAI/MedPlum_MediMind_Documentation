FHIR Resource Creation Flow

When a registration is submitted, the following FHIR resources are created:

sequenceDiagram participant U as User participant F as Form participant S as Service participant M as Medplum U->>F: Submit Registration F->>S: createPatient() S->>M: POST /Patient M-->>S: Patient ID F->>S: createEncounter() S->>M: POST /Encounter M-->>S: Encounter ID F->>S: createCoverage() S->>M: POST /Coverage M-->>S: Coverage ID S-->>F: Success F-->>U: Registration Complete
